# Tools
- front-end: React.js (create-react-app)
- back-end: Express.js
- mongooseDB, online: https://mlab.com

# Start 
### Server
```
cd server
yarn start
```

### Client
```
cd client
yarn start
```

### log in to add new post
```

password: 1234
```
